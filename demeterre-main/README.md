## HTTPS local pour telephone

Pour utiliser la camera sur Chrome mobile, lance le serveur Vite en HTTPS local :

```sh
npm run dev:https
```

Ensuite :

- sur ordinateur, ouvre `https://localhost:8080`
- sur telephone connecte au meme Wi-Fi, ouvre `https://ADRESSE_IP_DE_TON_PC:8080`

Notes importantes :

- le navigateur affichera un avertissement de certificat local la premiere fois
- sur ordinateur, accepte l'exception de securite pour continuer
- sur telephone, il faut parfois faire confiance au certificat local du PC pour que Chrome accepte vraiment la camera
- si le telephone refuse encore la camera, la solution la plus fiable reste une URL HTTPS publique de test
